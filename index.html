cheaptech

the cheaper it is, the less you'll care when it breaks.(tm)

  • cheappod for Mac OS X
    download songs from your iPod back to a computer.
  • cheapdns domain authority DNS server
    IT'S SECURE BECAUSE IT'S HARD CODED(tm);
  • cheaptxt t-mobile text messaging web-form submitter
    This one could be instructive if you needed an example of basic LWP usage. and possibly a good example of silent, unhelpful error handling.
  • Grab for Windows
    Loosely modeled off of the Grab.app utility on Mac OS X. Allows you to quickly and easily "grab" a section of the screen (i.e. the "PrtScn" key with some extra features).
    Download includes source code (VB6 project) and an executable Grab.exe. To use, run Grab.exe and drag a rectangular selection over your screen. You should then get a popup menu with some options.
    caveats: doesn't support multiple monitors very well (by which I mean 'at all'). The way the selection rectangle is implemented is pretty cheap. Also it doesn't look like it works in XP without CommDlg32.ocx, which you may need to download.
  • F11 for Windows
    Reveal the desktop. Mac OS X users will find this familiar and probably useful.
